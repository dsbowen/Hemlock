Input group base

hemlock.InputGroup

A base class for questions with input groups.

Attributes: append : str

Text appended to the input field. Specifically, the text of the div.input-group-append tag.

prepend : str

Text prepended to the input field. Specifically, the text of the div.input-group-prepend tag.
